# meme-generator
A meme generator

Download html2canvas.js from [here](http://html2canvas.hertzen.com/).
Move it into the static/js folder.
Point to this file from within the HTML:
```html
<script src="static/js/html2canvas.js"></script>
```

## Local Development

Visual Studio Code.
Live Server extension.
